## Documentation Principles

- Formal proofs are separated from computational evidence.
- Every public claim is traceable to supporting artifacts.
- Dependencies are explicitly documented.
- Reproducibility is treated as a first-class design objective.
- Limitations and open problems are documented alongside verified results.
